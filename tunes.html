GENIE comprehensive model configurations and tunes

This section contains a summary description of public GENIE tunes and comprehensive model configurations and provides links to tuning outputs (best-fit parameters, covariances) as well as detailed quantitative characterizations of tunes against an extensive collection of data.

Details for the physics configurations and tunes, as well as selected data/MC comparisons may be found in the Physics & User manual. The complete list of data/MC comparisons characterizing each tune will be linked in the table below in due time. Please note that GENIE produces many more tunes than it makes available for public use (including several partial tunes of datasets shown below). If you have a specific requirement, please contact the GENIE team.

Tunes marked as `stock', include parameter values from a mixture of past GENIE tunes, external tunes, or reasonable first estimates for new modelling elements that is too complex to note in this summary table. They form the starting point for a branch of more refined GENIE tunes and they are, themselves, valid and supported GENIE tunes.

To familiarize yourself with the GENIE naming conventions for comprehensive physics configurations and physics tunes, please see here.
